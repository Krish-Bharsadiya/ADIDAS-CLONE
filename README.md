# 👟 Adidas Clone

A responsive **Adidas Landing Page Clone** built using **HTML, CSS**. This project recreates the modern Adidas website UI with a clean design, responsive layout, and interactive user experience.

---

## 📖 About the Project

This project is a frontend clone of the official Adidas website created for learning and practicing frontend web development skills.

It focuses on:
- Responsive Web Design
- Modern UI/UX
- HTML5 Semantic Structure
- CSS3 Styling & Animations
---

## ✨ Features

- 🏠 Responsive Homepage
- 🎨 Modern Adidas-inspired UI
- 📱 Mobile-Friendly Design
- 🛍️ Product Showcase Section
- 🖼️ High-Quality Images & Icons
- ⚡ Smooth Hover Effects
- 🎯 Clean Navigation Bar
- 💻 Cross Browser Compatible

---

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Krish-Bharsadiya/ADIDAS-CLONE.git
```

### Go to project directory

```bash
cd ADIDAS-CLONE
```

### Open the project

Simply open **index.html** in your browser.

Or use VS Code Live Server.

---

## ▶️ Running the Project

Using Live Server:

1. Open the project in VS Code.
2. Install the **Live Server** extension.
3. Right-click on `index.html`.
4. Click **Open with Live Server**.
